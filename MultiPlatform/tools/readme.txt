				MultiPlatform CCDL的资源打包工具

  该工具可以将使用MultiPlatform SDK编译的CC程序与资源文件打包为一个带资源的CC文件.

					使用方法

packer <输出文件名,CC格式> <输入文件名,CC格式> <资源文件的目录> <打包格式,请输入CCDL>

以打包仙剑奇侠传为例,
d:\palres目录里面为仙剑所需要用到的所有资源,可执行文件为d:\pal.cc.
现在要将pal.cc和所有的资源打包成"仙剑奇侠传.cc",
执行命令:
packer d:\仙剑奇侠传.cc d:\pal.cc d:\palres CCDL

就可以生成一个大文件"仙剑奇侠传.cc",它就可以单独发布了.

					资源的访问方式

    资源文件被映射到程序所在的目录下,只要在开发的时候,所有资源文件都放在程序目录下,包括子目录也行,使用标准的文件系统访问即可.
    当资源文件在当前目录下存在的时候,优先使用真实存在的文件,当找不到文件的时候,才从资源包中获取相应的文件.

    该打包工具需要和shell库结合使用才行,因此当发现功能不正常时,请联系我并确认你使用的shell库是最新版本.旧程序必须重新使用新的库重编译.
